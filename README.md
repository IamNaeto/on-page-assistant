# On-Page-Asistant

- A transparent, moving, and dynamic on-page assistant situated at the bottom right of the page that transits next to a button clicked on and provides helpful information to users concerning what the button does.
    - This project is HNGx stage 5 task 2.

## Demo

Live Link :: [Go Live](https://on-page-web-assistant.vercel.app/).

## Features

- Feature 1: A transparent, moving, and dynamic on-page assistant situated at the bottom right of the page
- Feature 2: Once any of the button in the page is clicked on, it transits next to the button and explain what it does.
